# Swing Backend API\n\nComplete Node.js/Express backend for the Swing training and job placement platform.\n\n## Features\n\n- User authentication with OTP verification\n- Role-based access (Trainee, Trainer, Admin)\n- MongoDB database with complete schema\n- Email notifications\n- Trainee management\n- Trainer assignment system\n- Job recommendations\n- Organization management\n- Admin dashboard endpoints\n\n## Tech Stack\n\n- Node.js + Express\n- MongoDB with Mongoose\n- JWT Authentication\n- Bcrypt for password hashing\n- Nodemailer for email\n- Express Validator for validation\n\n## Installation\n\n1. Install dependencies:\n```bash\nnpm install\n```\n\n2. Setup environment variables:\n```bash\ncp .env.example .env\n```\n\nEdit `.env` with your configuration:\n- MongoDB URI\n- JWT Secret\n- Email credentials\n\n3. Start development server:\n```bash\nnpm run dev\n```\n\n4. Server runs on `http://localhost:5000`\n\n## API Endpoints\n\n### Authentication\n- `POST /api/auth/register` - Register new user\n- `POST /api/auth/verify-otp` - Verify OTP\n- `POST /api/auth/login` - Login user\n- `GET /api/auth/me` - Get current user\n\n### Trainee\n- `GET /api/trainee/:id` - Get trainee profile\n- `PUT /api/trainee/:id/profile` - Update trainee profile\n- `GET /api/trainee/:id/jobs` - Get recommended jobs\n\n### Trainer\n- `GET /api/trainer/:id` - Get trainer profile\n- `GET /api/trainer/:id/trainees` - Get assigned trainees\n- `POST /api/trainer/:trainerId/certify/:traineeId` - Certify trainee\n- `POST /api/trainer/:trainerId/fail/:traineeId` - Mark trainee as failed\n\n### Admin\n- `GET /api/admin/trainees` - Get all trainees\n- `GET /api/admin/trainers` - Get all trainers\n- `POST /api/admin/assign-trainee` - Assign trainee to trainer\n- `POST /api/admin/recommend-job` - Recommend job to trainee\n- `GET /api/admin/organizations` - Get all organizations\n- `POST /api/admin/organizations` - Create organization\n- `PUT /api/admin/organizations/:id` - Update organization\n- `GET /api/admin/jobs` - Get all jobs\n- `POST /api/admin/jobs` - Create job\n\n## Database Schema\n\n### Trainee\n- name, email, phone, password\n- existingSkills, skillsOfInterest\n- trainingStatus (pending, assigned, in-progress, certified, failed)\n- assignedTrainer, certificationDate\n- jobRecommendations, progress\n- isVerified, rejectionReason\n\n### Trainer\n- name, email, phone, password\n- expertise, rating\n- assignedTrainees, certifiedTrainees, failedTrainees\n- isVerified\n\n### Admin\n- name, email, password\n- role (admin, super_admin)\n- permissions, isActive\n\n### Organization\n- name, location, description\n- contactEmail, contactPhone, website\n- numberOfJobs, jobs array\n- createdBy, isActive\n\n### Job\n- title, description, skillsRequired\n- organization, salary, location\n- jobType, postedBy\n- recommendedTo, applicants array\n- isActive\n\n### Notification\n- recipient, recipientModel\n- type, title, message\n- relatedTo, isRead, readAt\n\n### OTP\n- email, otp, attempts\n- expiresAt (10 minutes)\n\n## Environment Variables\n\n```\nPORT=5000\nMONGODB_URI=mongodb://localhost:27017/swing\nJWT_SECRET=your_secret_key\nEMAIL_SERVICE=gmail\nEMAIL_USER=your_email@gmail.com\nEMAIL_PASSWORD=your_password\nNODE_ENV=development\n```\n\n## OTP System\n\n- 6-digit OTP sent to email\n- Maximum 3 attempts before 5-minute lockout\n- OTP expires in 10 minutes\n- Automatic cleanup of expired OTPs\n\nAuthor: Swing Development Team\nVersion: 1.0.0\n
